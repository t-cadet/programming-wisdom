The Protocol

The following protocol was used to gather the data for this test:

  1. Source Selection: Search for "C" on GitHub and browse the first search results.
  2. Human Snippets: Find 10 functions; remove comments and format the code to eliminate stylistic clues.
  3. AI Generation: For each selected human function, provide only the function signature to a modern LLM; ask it to implement the function.
  4. Presentation: For each of the 10 snippets in the test, the app randomly chooses whether to show you the Human version or the AI version.
